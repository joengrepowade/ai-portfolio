## Projects

| Project | Description | Stack |
|---------|-------------|-------|
| video-multimodal-ai | Video understanding & multimodal embedding (video + text + audio) | PyTorch, CLIP, TimeSformer |
| ml-inference-optimization | Model quantization, TensorRT, ONNX export, latency benchmarking | ONNX, TensorRT, torch.compile |
| ray-distributed-training | Distributed training & hyperparameter tuning with Ray Train + Ray Tune | Ray, PyTorch |
| kubernetes-ml-serving | K8s-based ML model serving with autoscaling and health monitoring | Kubernetes, FastAPI, Docker |
